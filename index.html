About

F.I.R.E. stand for Fast, Integrated, Reliable & Engaging.

These are the four pillars of UX, DX (Developer eXperience) & quality everyone should care about when creating Web Apps. Advocating for their understanding and adoption (and therefor for Web Developers) is one of my main focuses.

Wof.show is where I put pretty much everything I created for this purpouse:

  • my blog (on dev.to)
  • the Wof Show Console, where you'll find a bunch of information about the Wof Show, including my talks and slides
  • talk abstracts you can read if you want some more information about a talk you attended or you want me to present during an event
  • videos (on Youtube)
  • and of course, all of these and more are open source and available on Github

Please, do not hesitate to send me a DM on Twitter or an email, or to open an issue on Github if you have any question or comment.
